---

### Stack:

- TypeScript
- React
- Next.js
- styled-components
- SWR
- JEST

## Getting Started

1.Install dependencies

```bash
yarn
```

2.Rename .env.dist to .env.local

3.Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
